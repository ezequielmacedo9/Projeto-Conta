# Sistema de Conta Bancária em Java

Projeto desenvolvido em Java simulando o funcionamento básico de uma conta bancária.

## Funcionalidades
- Consulta de saldo
- Depósito
- Saque
- Transferência entre contas
- Menu interativo no terminal

## Tecnologias utilizadas
- Java
- Programação Orientada a Objetos (POO)

## Como executar o projeto
1. Compile os arquivos `.java`
2. Execute a classe principal pelo terminal ou IDE

## Observações
Projeto desenvolvido para fins acadêmicos, com foco em Java e lógica de programação.
